# github-dark-theme

A dark theme for Refined Github and Zenhub

## Instructions:

1. Install the [Refined Github](https://github.com/sindresorhus/refined-github) Chrome plugin.
2. Open the toolbar.
3. Paste the contents of style.css into the CSS overrides box.
